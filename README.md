# Spreadsheet Assistant

An AI-powered spreadsheet assistant that enables users to interact with Excel files using natural language. Upload a spreadsheet and ask questions about your data, perform calculations, generate summaries, and gain insights without writing formulas or SQL.

## Features

- Upload Excel spreadsheets (`.xlsx`, `.xls`)
- Chat with your spreadsheet in natural language
- Answer questions about tabular data
- Perform mathematical calculations on spreadsheet data
- Generate summaries and descriptive statistics
- Search and retrieve relevant information from large spreadsheets
- Powered by Large Language Models (LLMs)
- Interactive Vue.js frontend with a Python backend

---

## Example Questions

- What is the total revenue for 2024?
- Which employee had the highest sales?
- Calculate the average salary by department.
- Show me the top 10 customers by spending.
- How many orders were placed in March?
- What trends do you observe in monthly revenue?
- Which products are underperforming?
